=head1 NAME

pkgrm - remove software packages

=head1 SYNOPSIS

B<pkgrm> [I<OPTION>] I<PKGNAME>

=head1 DESCRIPTION

B<pkgrm> is a package management utility, which removes/uninstalls a
previously installed software packages.

=head1 OPTIONS

Mandatory arguments to long options are mandatory for short options
too.

=over 8

=item B<-r>, B<--root>=I<DIR>

Specify an alternate root directory.  This should be used if you want
to remove a package from a temporary mounted partition, which is
"owned" by another system.  By using this option you not only specify
where the software is installed, but you also specify which package
database to use.

Default: C</>.

=item B<-v>, B<--verbose>

Explain what is being done.

=item B<-V>, B<--version>

Print version and exit.

=item B<-h>, B<--help>

Print help and exit.

=back

=head1 FILES

=over 8

=item F</var/lib/pkg/db>

Database of currently installed packages.

=back

=head1 SEE ALSO

L<pkgadd(8)>, L<pkginfo(1)>

=cut
